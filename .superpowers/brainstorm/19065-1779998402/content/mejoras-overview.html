¿Qué mejoraría de la web?

Análisis del sitio actual — seleccioná las mejoras que más te interesan

A

Menú móvil (hamburger)

En mobile no hay forma de navegar — los links del nav están ocultos con hidden lg:flex. Sin menú hamburger el usuario tiene que scrollear a ciegas.

CRÍTICO — afecta usabilidad
B

Textos en el hero carousel

El código ya tiene kicker, título y descripción definidos para cada slide — pero el componente HeroCarousel solo muestra las imágenes. Esos textos se están desperdiciando. Un overlay de texto bien diseñado haría el hero mucho más potente.

IMPACTO VISUAL ALTO
C

Animaciones al hacer scroll

Las secciones aparecen de golpe. Con fade-in suaves cuando cada sección entra en viewport, el sitio se siente mucho más premium y profesional.

DISEÑO PREMIUM
D

Refactor del código (index.tsx)

Todo el sitio está en un único archivo de ~650 líneas. Separar en componentes (HeroCarousel, ExperienceCarousel, ReservationForm, MenuSection) y archivos de datos (menu-data.ts, slides-data.ts) hace el código más fácil de mantener y escalar.

CALIDAD DE CÓDIGO
Podés seleccionar varias. Mi recomendación: A + B primero (mayor impacto), luego C y D.
