ANTES — Todo en un archivo

Estructura actual

~650 líneas en un solo archivo. Difícil de mantener.

src/
routes/
index.tsx   ← 650+ líneas
  # datos del menú (132 líneas)
  # datos del hero (40 líneas)
  # componente Index
  # HeroCarousel
  # ExperienceCarousel
  # ReservationForm
❌ Monolito. Cada vez que se edita algo (menú, hero, formulario) hay que navegar por todo el archivo. Los datos mezclados con los componentes hacen que sea frágil de tocar.
DESPUÉS — Separado por responsabilidad

Estructura propuesta

Cada archivo tiene una sola responsabilidad. Fácil de editar y escalar.

src/
data/
menu-data.ts   ← carta del restaurante
slides-data.ts  ← hero + experience slides
components/
HeroCarousel.tsx
ExperienceCarousel.tsx
ReservationForm.tsx
MobileMenu.tsx
routes/
index.tsx   ← ~120 líneas (solo layout)
✅ Separación clara. Para cambiar la carta: editás menu-data.ts. Para el hero: slides-data.ts. El archivo index.tsx queda en ~120 líneas, fácil de leer de un vistazo.
